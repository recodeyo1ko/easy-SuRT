# かんたん SuRT

- SuRT(Surrogate Reference Task)を簡単に実施できるアプリケーション
- 画面の中から
- 画面サイズは iphone12mini を想定

## 使用方法

- iphon
- https://easy-su-rt.vercel.app/ にアクセス
- 「ホーム画面に追加」を選択後、ホーム画面から起動

## 使用技術

- React-v18.2.0
- MUI-v5.14.20

## Usage

### local

- `npm install`
- `npm start`

### docker

- `docker-compose run front npm install`
- `docker-compose up`
